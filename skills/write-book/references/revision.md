# 长篇修订协议

## 目录

- 建立修订基线
- 五层修订
- 影响分析
- 完成判据

## 建立修订基线

先冻结一份可回退的当前稿，记录本轮修订目标和明确不处理的内容。一次只选择少量可验证目标，例如“修复中段因果链”或“统一术语”，不要用“整体提升”作为唯一要求。

## 五层修订

### 1. 书籍合同

检查目标读者、核心承诺、类型、范围和结尾是否一致。删除服务于另一本书的内容。

### 2. 全书结构

只读 `outline.md`、章节索引和全部章节摘要，检查顺序、重复、缺口、节奏和承诺兑现。此阶段不润色句子。

### 3. 章节与场景/论证

逐章核对章节合同。小说检查目标、阻力、选择、代价和状态变化；非虚构检查问题、主张、证据、限制和推论。

### 4. 连续性与事实

核对 `canon.md`、`continuity.md` 和 `research.md`。统一时间线、人物/概念状态、术语、数字、引文和来源。把每个冲突处理为：修改正文、更新事实层，或保留并解释。

### 5. 文风与校对

最后处理段落顺序、句子节奏、重复词、空泛表达、语法、标点和格式。抽样朗读，检查声音是否符合 `style.md`。

## 影响分析

跨章节变更前，写出：变更内容、事实源位置、直接影响章节、通过伏笔/引用/术语产生的间接影响、需要刷新的摘要和验收方式。按影响清单逐项完成，不要进行无记录的全局替换。

## 完成判据

本轮修订只有在以下条件同时满足时才完成：

- 目标问题有前后对比或检查证据；
- 所有修改章节的摘要已刷新；
- 连续性和研究账本没有遗留的无说明冲突；
- 章节索引反映真实状态；
- 新导出物通过格式和目视检查。
