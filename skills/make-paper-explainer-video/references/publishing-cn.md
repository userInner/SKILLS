# China short-video publishing package

Use this reference when packaging for Douyin, Xiaohongshu, or WeChat Channels.

## Verify before packaging

Platform limits, content declarations, and UI labels change. Check current official rules or the live publishing form before stating a limit as fact.

At minimum verify:

- title and description length;
- accepted aspect ratio, codec, duration, and file size;
- cover ratio and safe crop;
- AI-generated or synthesized content declaration;
- original/reposted, marketing, fictional, or personal-opinion labels;
- whether first comments can be pinned.

## Common deliverables

- one verified 9:16 MP4;
- clean 9:16 cover for Douyin;
- clean 3:4 cover for Xiaohongshu and WeChat Channels;
- platform-specific title;
- description/body;
- 4-7 relevant hashtags;
- first comment containing either a limitation or discussion question;
- source identifier such as DOI or arXiv number;
- explicit note describing which elements were AI-generated;
- upload checklist.

## Platform voice

### Douyin

- Put the conflict or question in the title.
- Keep the description compact and front-load the human case or main result.
- Put detailed sampling and causality boundaries in the description or first comment.
- Keep the title below the current verified limit; if not verified, use no more than 30 characters as a conservative default.

### Xiaohongshu

- Use a searchable, conversational title.
- Expand the reported cases and practical checklist in the body.
- Use short paragraphs and numbered points.
- State the paper status and limitations before the source line.

### WeChat Channels

- Use a restrained, shareable first sentence.
- Connect the result to work, learning, parenting, or another recognizable decision context without overstating generalization.
- Keep the description shorter than the Xiaohongshu body.

## AI disclosure

If any narration, image, video, or text is generated or synthesized by AI:

- select the platform's current AI-content declaration;
- state the affected component precisely, such as “视频配音由本地 AI 语音模型生成”;
- do not imply paper screenshots, measured data, or participant reports were AI-generated when they were not;
- preserve any required visible or metadata label.

## Public-submit boundary

Preparing files, covers, and form text does not authorize public posting. Upload or fill fields only when the user requests account actions. Ask for confirmation at the final public-submit step.
